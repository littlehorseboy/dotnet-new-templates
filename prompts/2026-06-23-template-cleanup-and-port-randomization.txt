了解！你是用 .NET dotnet new 的 template engine（template.json），不是 VSIX。
這個情境下，隨機 port 可以用 symbols + 後處理 來做：

template.json 裡定義一個 symbol
json"symbols": {
  "HttpsPort": {
    "type": "generated",
    "generator": "port",
    "parameters": {
      "low": 5000,
      "high": 5999
    },
    "replaces": "5001"
  }
}
然後你的 launch.json 裡就直接寫死 5001（當作佔位符）：
json{
  "url": "https://localhost:5001"
}
dotnet new 執行時會自動把所有 5001 換成隨機產生的 port。
